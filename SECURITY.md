# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Godel, please report it responsibly.

**Email:** godel@abrahamtoledo.dev

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will acknowledge receipt within 72 hours and aim to provide a fix or mitigation plan within 14 days.

**Please do not open a public GitHub issue for security vulnerabilities.**
